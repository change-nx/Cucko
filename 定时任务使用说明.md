# 定时任务使用说明

## 问题解决

之前定时任务发消息比较麻烦，因为发送函数依赖事件上下文。现在我们已经创建了独立的发送函数，不再依赖事件上下文。

## 新增文件

1. `function/other/CronHelper.php` - 定时任务辅助函数
2. `example_cron_qq.php` - QQ 适配器示例
3. `example_cron_onebot.php` - OneBot 适配器示例
4. `plugin/测试/cron_example/` - 插件示例
5. `定时任务使用说明.md` - 本文档

## 主要改进

### 1. 独立发送函数

#### QQ 适配器
```php
// 在定时任务脚本中使用
定时_QQ_群消息($适配器ID, $群号, $内容);
定时_QQ_私消息($适配器ID, $用户ID, $内容);
```

#### OneBot 适配器
```php
// 在定时任务脚本中使用
定时_OB_群消息($适配器ID, $群号, text("消息"), image("图片"));
定时_OB_私消息($适配器ID, $用户ID, text("消息"));
```

### 2. 定时任务管理

```php
// 添加任务
定时_添加($任务ID, $时间格式, $脚本路径);

// 删除任务
定时_删除($任务ID);

// 查看任务
$列表 = 定时_列表();
```

## 支持的时间格式

- `每分钟`
- `每小时`
- `每天凌晨` / `每天午夜`
- `每天8点` / `每天8点30分`
- `每5分钟`
- `每2小时`
- `每周一8点`
- `每月1号8点`
- `2026-05-20 14:30` (一次性任务)

## 使用步骤

### 方法一：直接创建定时任务脚本

1. 创建一个 PHP 脚本（如 `my_cron.php`）
2. 在脚本中使用 `定时_QQ_群消息()` 或 `定时_OB_群消息()` 发送消息
3. 使用 `定时::添加()` 或 `定时_添加()` 添加到 crontab

示例：
```php
<?php
// my_cron.php
require_once __DIR__ . '/function/other/CronHelper.php';

定时_QQ_群消息('your_adapter_id', '123456789', '这是定时消息！');
```

### 方法二：使用示例插件

1. 加载 `cron_example` 插件
2. 在群内发送：
   - `定时测试` - 查看帮助
   - `定时添加` - 添加每日8点提醒
   - `定时删除` - 删除提醒
   - `定时列表` - 查看任务

## 代码示例

### QQ 适配器完整示例

```php
<?php
require_once __DIR__ . '/function/other/CronHelper.php';

$适配器ID = 'your_adapter_id';
$群号 = '123456789';
$时间 = date('Y-m-d H:i:s');

$消息 = "【每日提醒】\n\n";
$消息 .= "现在是 {$时间}\n";
$消息 .= "祝您今天愉快！";

定时_QQ_群消息($适配器ID, $群号, $消息);
```

### OneBot 适配器完整示例

```php
<?php
require_once __DIR__ . '/function/other/CronHelper.php';

$适配器ID = 'your_adapter_id';
$群号 = 123456789;

$消息 = text("【定时播报】\n\n");
$消息 .= text("现在是 " . date('Y-m-d H:i:s'));

定时_OB_群消息($适配器ID, $群号, $消息);
```

### 在插件中添加定时任务

```php
<?php
// 在插件中添加定时任务
if (消息 == '添加任务') {
    // 1. 先创建定时任务脚本
    $脚本内容 = '<?php
require_once __DIR__ . \'/../../function/other/CronHelper.php\';
定时_QQ_群消息(\'' . adapter . '\', \'' . 群号 . '\', \'定时消息！\');
';
    $脚本路径 = __DIR__ . '/my_task.php';
    file_put_contents($脚本路径, $脚本内容);
    
    // 2. 添加到定时任务
    $结果 = 定时::添加('my_task_id', '每天8点', $脚本路径);
    
    if ($结果['success']) {
        群(群号, text("任务添加成功！"));
    }
}
```

## 注意事项

1. 脚本路径必须使用**绝对路径**
2. 适配器ID 要正确配置，并且 config 数据库中有该适配器的配置
3. 确保 PHP 有足够权限执行 crontab 命令
4. 定时任务ID 不能重复
